# STYLiSH

STYLiSH is an apparel e-commerce RWD website built with plain JavaScript.

Link : https://nycteachen-stylish.web.app/

Login : Facebook

---

## **Technologies**

### **Front-End**

- HTML
- CSS
- JavaScript(ES6+)

### **User Interface**

- Responsive Web Design(RWD)
- Flex and Grid Layout

### **Third-Party Library**

- Firebase
- Facebook
- Tappay
- Google Analitics

### **Networking**

- AJAX using Fetch
- RESTful API integrating

### **Persistent Store**

- Local Storage

### **Tools**

- ESLint
- Prettier
